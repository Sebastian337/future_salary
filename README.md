# 1. Описание проекта Future Salary

Набор скриптов для сбора статистики вакансий программистов в Москве с сайтов HeadHunter и SuperJob. Скрипты собирают информацию о количестве вакансий и средних зарплатах по популярным языкам программирования, выводя результат в виде удобных таблиц.

## 2. Требования к окружению

- ОС: Linux, macOS, Windows
- Python: версия 3.8 или выше
- Доступ к интернету для запросов к API
- Ключ API SuperJob (обязательно)
- HeadHunter API не требует авторизации

## 3. Как установить

### Установите зависимости:
```bash
pip install -r requirements.txt
```
### Настройте переменные окружения:

Создайте файл .env в корне проекта и добавьте туда свой ключ.
Ключ SuperJob API можно получить при регистрации приложения: https://api.superjob.ru/register
```txt
SUPERJOB_SECRET_KEY=v3.r.ваш_ключ_сюда
```
## 4. Использование скриптов

Основной скрипт для сбора статистики:

```bash
python main.py
```

Скрипт автоматически:

* Собирает вакансии программистов в Москве за последний месяц
* Обрабатывает данные с HeadHunter
* Обрабатывает данные с SuperJob
* Рассчитывает средние зарплаты по каждому языку
* Выводит две сравнительные таблицы

## Примечание:

* Скрипт анализирует вакансии только в Москве
* Учитываются только вакансии за последние 30 дней
* Учитываются только вакансии с зарплатой в рублях
* Между запросами установлены паузы (0.3 сек) для предотвращения блокировки
* SuperJob требует обязательной регистрации приложения и получения ключа API
* HeadHunter API работает без авторизации
